# tic-tac-toe
Создание игры
 Создать обьект: new Tic_tac(`размер сетки`,`условие для победы`)
 p.s Наличие атрибутов не обязательно, если есть принадлежат типу число, в противном случае создается сетка 3x3, условие для победы 3
